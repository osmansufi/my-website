12/03/14 - Blog Post #3

Boy this was some assignment. That is, this website you see before you. But I'm glad I did it. Every now and then I surprise myself and actually enjoy doing work. In fact, I intend to continue developing my web developer skills over the winter break and on to the next semester. But enough of that, onto the problems.

The major problem I confronted in this endeavor was my own ignorance of web development best practices. Though I have become more acquainted with them as a result of working on this assignment I still have much to learn. One specific example of a huge problem I confronted was that I initially designed the structure of every webpage on my website with tables. I learned, just in the nick of time, that divs and selectors are a far superior way of organizing a website-and I made the necessary changes accordingly.

In addition to this problem I had to also contend with things not displaying the same, or correctly, across different browsers. In Internet Explorer, for example, the menu on the left, with the links, was not showing a background color despite the fact that the CSS was written to include a background color. This color showed up in Firefox and Chrome on both the desktop and mobile versions of the website. It was only in Internet Explorer that it was not showing up. I resolved the issue by making a setting for establishing the color of visited links (I set the color to the same as non-visited, non-hovered over links). For some reason this seems to have resolved the issue.

In spite of these difficulties, and many others, I still managed to complete the website. But I don't intend to stop here-I intend to improve on it and to add to the website in the coming weeks and months.
